# 마리오 게임 모작

## 개요
API 활용을 위한 간단한 모작 게임

## 기술 스택
- WIN32 API
- VS
- C++

## 구현 목표
- 마리오 게임의 기본적인 움직임
- 캐릭터 이동을 통한 맵 이동
- 점프를 통한 굼바 사냥
- 몬스터 생성
- 간단한 엔딩 애니메이션
- 랭킹 리스트
